🎯 למה פונקציות טהורות?

הלוגיקה ב-gameLogic.ts ו-ai.ts מקבלת קלט ומחזירה פלט, בלי state ובלי DOM. לכן אפשר לבדוק אותה ישירות ב-Jest — למשל לשחק משחק מלא מול ה-minimax ולוודא שהוא לעולם לא מפסיד.

🔁 זרימת מהלך

לחיצה על משבצת ← placeMark מעדכן את הלוח ומעלה turnId ← הטיימר מתאפס ← אם תור המחשב, effect מפעיל את getAiMove אחרי השהיה קצרה.

דוגמה: הליבה הטהורה — calculateWinner

/** מחזיר את המנצח ואת הקו המנצח, או null אם אין מנצח עדיין. */
export function calculateWinner(squares: SquareValue[]): WinResult | null {
  for (const line of WINNING_LINES) {
    const [a, b, c] = line;
    if (squares[a] && squares[a] === squares[b] && squares[a] === squares[c]) {
      return { winner: squares[a] as Player, line: [a, b, c] };
    }
  }
  return null;
}

דוגמה: ה-AI הבלתי-מנוצח — minimax

// קל = אקראי · בינוני = נצח⇐חסום⇐אקראי · קשה = minimax
export function getAiMove(squares, aiMark, difficulty): number {
  switch (difficulty) {
    case 'easy':   return randomMove(squares);
    case 'medium': // קח ניצחון, אחרת חסום, אחרת אקראי
      return findWinningMove(squares, aiMark)
        ?? findWinningMove(squares, opponentOf(aiMark))
        ?? randomMove(squares);
    case 'hard':   return minimax(squares, aiMark, aiMark, 0).move;
  }
}

פיצ'רים

הפרויקט התפתח בשלושה גלים: המשחק הבסיסי, הטיימר ורמות הקושי, ולבסוף דף הבית ומצב המחשב. כל פיצ'ר תוכנן עם בדיקות משלו.

🎮 משחק בסיסי (PvP)

לוח 3×3, החלפת תורים אוטומטית, זיהוי ניצחון בשורה/עמודה/אלכסון, זיהוי תיקו, והדגשת הקו המנצח. כפתור "משחק חדש" מנקה את הלוח ושומר את ההגדרות.

👤 שמות שחקנים

כל שחקן מזין את שמו, והשם מופיע בתצוגת התור ובהודעת הניצחון. שדה ריק נופל לברירת מחדל ("שחקן X" / "שחקן O").

🤖 נגד מחשב (PvC)

השחקן האנושי הוא X (ראשון), המחשב הוא O ומגיב מיד אחרי השהיה קצרה לתחושה טבעית. חוזק ה-AI מתאים את עצמו לרמת הקושי שנבחרה.

🧠 AI לפי רמת קושי

קל — מהלך אקראי. בינוני — מנצח אם אפשר, אחרת חוסם את היריב, אחרת אקראי. קשה — אלגוריתם minimax שמשחק מושלם ולעולם לא מפסיד.

⏱️ טיימר לכל מהלך

פס התקדמות מתרוקן ככל שהזמן אוזל (אדום מתחת ל-3 שניות). אם הזמן נגמר — השחקן מאבד את התור והוא עובר ליריב. ב-PvC הטיימר רץ רק בתור האדם.

🎚️ שלוש רמות קושי

הקושי קובע גם את משך הזמן לכל מהלך: קל = 15 שניות, בינוני = 10, קשה = 5 — וגם את חוזק ה-AI במצב נגד מחשב.

🏠 דף בית

לפני המשחק בוחרים מצב (שחקן/מחשב), רמת קושי ושמות. הבחירות נשמרות בחזרה מהמשחק. הניווט מבוסס state פנימי (screen: 'home' | 'game') ללא router.

♿ נגישות

הלוח מסומן ב-role="grid", הסטטוס ב-role="status", הטיימר ב-role="timer" עם aria-label, וכל הבוררים עם aria-pressed.
